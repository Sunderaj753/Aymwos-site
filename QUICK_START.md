## Customization Guide

### Change Company Name/Content
Edit `index.html`:
- Line 7: Website title
- Line 23: Logo/Brand name
- Update all service descriptions
- Update contact information

### Change Colors
Edit `css/styles.css` (top of file):
```css
:root {
    --primary-color: #667eea;      /* Purple */
    --secondary-color: #764ba2;    /* Dark Purple */
    --accent-color: #f093fb;       /* Pink */
}
```

## Popular Customizations

### Add Contact Form Functionality
Use Formspree (free):
1. Go to https://formspree.io
2. Sign up with email
3. Add this to form tag in index.html:
   ```html
   <form class="contact-form" action="https://formspree.io/f/YOUR_CODE" method="POST">
   ```

### Add Google Analytics
1. Get tracking ID from https://analytics.google.com
2. Add tracking code to index.html head

### Add More Images
Replace the gradient backgrounds in portfolio items with real images:
```html
<!-- Instead of gradient background: -->
<div class="portfolio-image" style="background: url('path/to/image.jpg') center/cover;">
</div>
```

### Add Google Map to Contact
Embed in contact section:
```html
<iframe src="https://www.google.com/maps/embed?..." width="100%" height="400"></iframe>
```

## Common Issues & Fixes

### Site doesn't load?
- Wait 5-10 minutes for GitHub Pages to build
- Clear browser cache (Ctrl+Shift+Delete)
- Check DNS at https://dnschecker.org

### Domain not working?
- DNS changes take 24-48 hours
- Verify DNS records in your registrar
- Check GitHub Settings → Pages → custom domain

### Styling looks wrong?
- Hard refresh: Ctrl+F5
- Check CSS file is being loaded (F12 Developer Tools)
- Verify file paths are correct
