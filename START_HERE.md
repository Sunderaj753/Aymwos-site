## 🌐 DEPLOYMENT IN 3 EASY STEPS

### STEP 1: Create GitHub Account & Repository
1. Go to https://github.com
2. Sign up or login
3. Click "New" to create repository
4. Name it: **YOUR_USERNAME.github.io**
   - Example: if your username is "aymwos", name it "aymwos.github.io"
5. Choose "Public"
6. Click "Create repository"

### STEP 2: Push Your Code to GitHub
Run these commands in PowerShell:

```powershell
cd C:\Users\sunde\OneDrive\Desktop\pyspark\Aymwos-site

git remote add origin https://github.com/YOUR_USERNAME/YOUR_USERNAME.github.io.git

git branch -M main

git push -u origin main
```

✅ Your website is now live at: https://YOUR_USERNAME.github.io

### STEP 3: Connect Your Custom Domain
1. Go to GitHub repository Settings → Pages
2. Add custom domain: `aymwostechnologies.com`
3. Save
4. Update DNS at your domain registrar (add A records pointing to GitHub Pages)
5. Wait 24-48 hours for DNS to propagate
6. Your site is now at: https://aymwostechnologies.com
